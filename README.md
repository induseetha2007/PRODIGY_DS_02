# PRODIGY_DS_02

## Task 2: Data Cleaning and Exploratory Data Analysis (EDA)

### Objective
Perform data cleaning and exploratory data analysis on the Titanic dataset to identify patterns, relationships, and insights related to passenger survival.

### Dataset
Titanic Dataset

### Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

### Project Workflow
- Imported and explored the dataset
- Checked for missing values and cleaned the data
- Removed duplicate records
- Performed descriptive statistical analysis
- Created visualizations including histograms, count plots, box plots, pair plots, and a correlation heatmap
- Analyzed relationships between passenger attributes and survival

### Files Included
- Task2.ipynb
- titanic.csv

### Outcome
Successfully cleaned and analyzed the Titanic dataset using exploratory data analysis techniques. The visualizations provided meaningful insights into passenger demographics, ticket class, age distribution, and survival patterns.
